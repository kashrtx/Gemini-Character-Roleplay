# Google Gemini Flash Character RolePlay

first you need to configure the chat by going into settings and adding your google gemini api key for flash 2.0 which you input and save at the top.

By utilizing Google Gemini 2.0 Flash you can  chat with your favourite characters. First give context such as the character and what anime they are from e.g. Light Yagami from Death Note aka smart. You just type whatever character you want and if you want you can ask gemini to improve your character context with an optimized prompt to create a perfect character context for you so that your character context has the proper personality and traits for the best chat.

After the character context has been set. You can start chatting with your character in the main message window. You can delete messages in the chat and the bot will understand whats in the chat. You can add multiple characters to chat with.
